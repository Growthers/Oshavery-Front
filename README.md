# Oshavery-Front: A Discord-like Chat tool  

## What is Oshavery?  

`Oshavery-Front` is Discord/Slack-like chat tool written in Next.js.  
`Oshavery-Front` は, Next.jsで書かれたDiscord/Slackライクなチャットツールのクライアントです.
